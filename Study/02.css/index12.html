CSS display 속성에 대해 알아보자!

  • display 속성은 화면에 어떻게 보여질지 결정하는 속성
  • html 태그는 기본적으로 block과 inline 요소 둘 중에 하나를 기본값으로 가짐

inline

  1. span, a, img 태그와 같이 내용물만큼(content)만 공간 할당됨
  2. content 만큼 공간을 할당하기 때문에 width, height, padding, margin 사용 어려움
  3. 줄바꿈 없이 바로 옆에 배치
d-inline1 클래스를 갖고 있는 div입니다.
d-inline2

block

  1. div, p, ul, ol 태그와 같이 한 줄을 다 차지함
  2. inline 요소와 달리 width, height, padding, margin 사용 가능
d-block1 클래스를 갖고 있는 div입니다.
d-block2

inline-block

  1. block 요소처럼 크기 지정을 하되
  2. inline 요소처럼 다른 내용들과 같은 줄을 공유하고 싶은 경우
  3. block과 inline 요소의 장점을 가짐
d-inline-block1 클래스를 갖고 있는 div입니다.
d-inline-block2

none

  1. 요소를 화면에 표시하지 않음
  2. 주의) display: none;은 요소가 차지하는 공간도 사라진다!!!
none

vs. visibility 속성

  1. display: none; 은 요소가 차지하는 공간이 사라지지만, visibility: hidden; 은 요소만 사라지고 차지하던 공간은 그대로 남는다
